# market_holidays

Custom maintained market holidays.

Currently supports XNYS and XTKS.

Holidays are based on [gerrymanoim/exchange_calendars](https://github.com/gerrymanoim/exchange_calendars) and references documented below.

## References

XNYS Holidays: https://www.nyse.com/markets/hours-calendars

XTKS Holidays: https://www.jpx.co.jp/english/corporate/about-jpx/calendar/

Spring & Autumn Equinox Day prediction: https://www.nao.ac.jp/faq/a0301.html
